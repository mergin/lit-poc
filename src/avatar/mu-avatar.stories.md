# mu-avatar Stories

## Basic Usage

```html
<mu-avatar initials="AB"></mu-avatar>
<mu-avatar
  src="https://i.pravatar.cc/100"
  alt="Avatar"
></mu-avatar>
```

## Sizes

```html
<mu-avatar
  initials="S"
  size="small"
></mu-avatar>
<mu-avatar
  initials="M"
  size="medium"
></mu-avatar>
<mu-avatar
  initials="L"
  size="large"
></mu-avatar>
```
