# mu-icon Stories

## Basic Usage

```html
<mu-icon name="check_circle"></mu-icon>
<mu-icon
  name="error"
  color="error"
></mu-icon>
<mu-icon
  name="star"
  color="warning"
></mu-icon>
```

## Sizes

```html
<mu-icon
  name="check_circle"
  size="small"
></mu-icon>
<mu-icon
  name="check_circle"
  size="medium"
></mu-icon>
<mu-icon
  name="check_circle"
  size="large"
></mu-icon>
```

## Colors

></mu-icon>

></mu-icon>
