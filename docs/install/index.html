Install

All components are distributed on npm under their respective package names, e.g. mu-avatar, mu-button, etc. Install them locally or use a CDN like unpkg.com.

Local Installation

npm i mu-avatar mu-button mu-card mu-icon

CDN

npm CDNs like unpkg.com can directly serve files published to npm. This works for standard JavaScript modules that browsers can load natively.

HTML

<script type="module" src="https://unpkg.com/mu-avatar?module"></script>
<script type="module" src="https://unpkg.com/mu-button?module"></script>
<script type="module" src="https://unpkg.com/mu-card?module"></script>
<script type="module" src="https://unpkg.com/mu-icon?module"></script>

JavaScript

import 'https://unpkg.com/mu-avatar?module';
import 'https://unpkg.com/mu-button?module';
import 'https://unpkg.com/mu-card?module';
import 'https://unpkg.com/mu-icon?module';
